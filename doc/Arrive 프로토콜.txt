=============== Client to Server ===========
create new room (Client to Server)
{
  "type" : "create_room",
  "name" : "jinseong's room",
  "password" : 1234,
  "lock" : true
}
enter room (Client to Server)
{
  "type" : "enter_room",
  "rommNum" : 2,
}
get out room
{
    "type" : "get_out_room",
    "roomNum" : 2
}
game start
{
    "type" : "game_start",
    "roomNum" : 2
}
change ready
{
    "type" : "change_ready",
}
change map
{
    "type" : "change_map",
    "roomNum" : 2,
    "name" : "firstMap"
}
lose hp
{
    "type" : "lose_hp",
    "roomNum" : 2
}

================ Server to Client
send entire roomList (Server to Client)
{
  "type" : "entire_roomList",
  "roomList" : [
    {
      "roomNum" : 1,
      "name" : "jinseong's room",
      "password" : 1234,
      "playerNum" : 2
    }
    ...
  ]
}

send character Position (Server to Client)
{
   "type" : "characterPosition",
   "x" : "(double 형)"
   "y" : "(double 형)"
}
